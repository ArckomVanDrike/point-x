schema semplificato del database per memorizzare i dati degli utenti e dei luoghi nel progetto "World X - Social dei Viaggi":

Tabella "Users" (Utenti):
Campo	Tipo	Descrizione
user_id	INT (PK)	ID univoco dell'utente (chiave primaria)
username	VARCHAR(50)	Nome utente
email	VARCHAR(100)	Indirizzo email dell'utente
password	VARCHAR(100)	Password criptata dell'utente
profile_pic	VARCHAR(255)	URL dell'immagine del profilo dell'utente
created_at	DATETIME	Data e ora di registrazione dell'utente

Tabella "Places" (Luoghi):
Campo	Tipo	Descrizione
place_id	INT (PK)	ID univoco del luogo (chiave primaria)
user_id	INT (FK)	ID dell'utente che ha inserito il luogo
name	VARCHAR(100)	Nome del luogo
description	TEXT	Descrizione del luogo
plus_code	VARCHAR(20)	Plus Code del luogo (es. V75V+8Q)
photo	VARCHAR(255)	URL dell'immagine del luogo
created_at	DATETIME	Data e ora di inserimento del luogo

Note:

    PK: Primary Key (Chiave Primaria) - Identifica univocamente ogni record nella tabella.
    FK: Foreign Key (Chiave Esterna) - Collega il record nella tabella "Places" all'utente corrispondente nella tabella "Users".
